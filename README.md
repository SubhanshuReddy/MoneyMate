This Expense Tracker is a simple and efficient web application designed to help users manage their daily finances with clarity and control. It allows users to record income and expenses, categorize transactions, and view real-time financial summaries.

The application automatically calculates total income, total expenses, savings, and current balance. Built-in analytics such as daily average spending, highest expense detection, and monthly comparison provide meaningful insights into spending habits.

All data is securely stored in the browser using local storage, ensuring privacy and persistence without the need for accounts or servers. The interface is clean, responsive, and includes features like dark mode, search, sorting, and PDF export for convenience.

This project is built using HTML, CSS, and JavaScript only, focusing on core web technologies to deliver a fast, lightweight, and user-friendly experience.
